本文使用的是 Selenium 模块，它是一个自动化测试工具，利用它我们可以驱动浏览器执行特定的动作，如点击、下拉等操作，对于一些 JavaScript 渲染的页面来说，此种抓取方式非常有效。

采用了 Chrome 浏览器配合 Selenium 工作，本文的 Python 版本是 3.7.2。
